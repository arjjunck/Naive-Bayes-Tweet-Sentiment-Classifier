# Naive-Bayes-Tweet-Sentiment-Classifier
This project implements a Multinomial Naive Bayes model from scratch in Python to classify tweets as positive or negative sentiment, achieving ~75% accuracy
Dataset: 400k positive (trainPos.txt) + 400k negative (trainNeg.txt) tweets

Tech Stack
Python 3.x | Pandas (data handling) | NumPy (log probs)
